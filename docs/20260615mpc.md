### 2.2 数据库扩展

Post 表新增字段：
- `coverImage String?` — 封面图 URL
- `mediaType String @default("text")` — text / image / album / video
- `images String?` — 相册多图（JSON 数组）
- `videoUrl String?` — 视频地址
- `tags String?` — 自选标签（JSON 数组）

### 2.3 前端——全局样式系统

- 字体系统：PingFang SC 优先，5 级字号 + 粗细分层
- 动画系统：弹簧缓动曲线 + 淡入/上浮/弹性入场
- 色彩变量：15 个 CSS 自定义属性，统一全站配色
- 骨架屏：shimmer 扫光动画
- Tab 页面淡入过渡

### 2.4 前端——广场页重构（小红书风格瀑布流）

**布局：** 双列瀑布流，JS 层 `splitWaterfall()` 按高度均衡分配左右列

**四种卡片类型：**
| 类型 | 封面 | 角标 |
|---|---|---|
| text | 9 种弥散渐变 + 左侧绿线 + 水印引号 | 左上分类标签 |
| image | 真实图片 | 左上分类标签 |
| album | 真实图片（首图） | 左上分类 + 右上胶囊数字 |
| video | 真实图片（封面） | 左上分类 + 右上圆形播放键 |

**视觉系统：**
- 卡片：`border-radius: 16rpx` + `overflow: hidden` + 弥散阴影 + 半透明细边
- 9 种 CSS `radial-gradient` 弥散渐变（暖橙/薄荷/海洋/薰衣草/玫瑰/天空系）
- 3 种宽高比（4:3 / 1:1 / 3:4）随机分配
- 统一底部栏：头像 + 昵称 + 时间
